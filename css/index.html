Introducción

Micro:bit es una pequeña pero poderosa placa de desarrollo diseñada para la enseñanza de programación y electrónica. A lo largo de este curso, aprenderás a utilizar Micro:bit para crear proyectos interactivos utilizando su conjunto de sensores, LEDs y comunicación inalámbrica.
